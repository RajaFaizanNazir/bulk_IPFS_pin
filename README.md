# bulk_IPFS_pin

### Testing on
* Ubuntu
* Windows 11
### Pre Req
* Python
* Pip (Package manager of Python)

> Guide to run the progrma

```sh
git clone https://github.com/RajaFaizanNazir/bulk_IPFS_pin.git
```

```sh
cd bulk_IPFS_pin
```

```sh
mkdir images
```

paste all images in the 'images' folder
 
 ```sh
 pip install -r requirements.txt
 ```
 
 ```sh
 python3 main.py
 ```
 or just run run.bat
